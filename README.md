# Circle Ecommerce App

![App Preview](demo_image/circle.png)

A dynamic **e-commerce application** built with Flutter during my internship project.  
The app focuses on seamless shopping experience, authentication, and real-time order management.

## Overview
Circle Ecommerce App enables users to browse products, authenticate with phone numbers and OTP, add items to cart, and place orders with flexible payment options.  
The project demonstrates **clean architecture**, scalable state management, and user-friendly design.

## Key Features
- **Phone Number Authentication + OTP Verification**  
- **Cart Management** (add/remove/update products in real time)  
- **Order Management** with status tracking  
- **Flexible Payments** (multiple payment options supported)  
- **Shimmer Loading Animations** for better UX  
- **Fast Image Loading** with caching  

## Architecture & Development Practices
- **Provider** for state management  
- **Dio** for networking & API integration  
- **IndexedStack & Mixins** to handle complex navigation  
- **Clean Architecture** for scalability and maintainability  

## Tech Stack
| Category | Technology |
|----------|------------|
| **Framework** | Flutter (Dart) |
| **State Management** | Provider |
| **Networking** | Dio |
| **Local Storage** | SharedPreferences |
| **Authentication** | Phone + OTP |

**Note**: This project was built as an **internship training application** to practice real-world Flutter concepts such as authentication, cart/order logic, and scalable app architecture.
